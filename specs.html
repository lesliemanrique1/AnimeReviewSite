Specifications

Overview

For my LAMP project, I will design an anime review website where registered users will be able to rate and review japanese anime shows. They will also be able to look up general information about the anime, such as the director, network, publisher and what source the anime is based on.

This website will be targeted to youthful audiences that watch anime and read manga. It will also helpt to attract more people to this form of media and to engage a community.

Navigation Bar

I will add general design features such as a logo and a navigation bar. The logo will have the name of the website Anime Battosai. The navigation bar will allow you to go to the Home page, About page the List and the Contact page. The List is what I will mainly focus on. Using PHP and MySQL, I will add a second navigation bar labeled A - Z where the user will be able to quickly access their anime by letter and search through the animes that start with the same letter. I may or may not add a search bar functionality.

Sort By

The Sort By feature will allow a second way for a user to see how the anime list is displayed. If the user selects Top-Rated the anime list will be sorted by the “star rating”. Users will be given the ability to give animes a star rating which allows other users to better identify how “good” the anime is. A maximum of 5 stars can be given to an anime and a minimum of 1 star can be given. The list displays the average of all the star ratings given to the anime by the user reviewers.

Users will also be able to view the list in alphabetical or reverse alphabetical order. And sort the list by the number of reviews they have. The titles with the most amount of reviews will be at the top of the list.

The final sorting option is a to present the anime list by genres using PHP and mySQL queries. It will appear similar to the following with the number of user reviews and average star ratings.

  • Action
  • Inuyasha
  • Rurouni Kenshin
  • Comedy
  • Ouran High School Host Club
  • The Devil is a Part-Timer!
  • Romance
  • Ouran High School Host Club

The Anime Profile

The next part is the anime profile. This includes the Information about the anime like the image - thumbnail- and the synopsis. At the top of the page the title will be included with the average user star rating. Below that I will add the image and synopsis. Next, the Genre’s. Most animes fit into more than one genre so there will be a list of hyperlinks users will click on that will take them to a list of animes of that genre. This will be identical to the sort by genre attribute from the list page only that it will automatically scroll the user to the genre name they clicked on. Afterwards there will be some general information about the anime such as production, seasons #of episodes. And afterwards information about the manga/ light novel or video game the anime is based upon.

The last piece is the review section. Users will be able to see the reviews of registered users which include the username, the date posted, star rating and a comment. For registered users, there will be a functionality that allows them to add comments to the anime profile. Such will be a button that says Add Comment that links to a submit form similar to the following:

Star Rating: 5

Comment: This anime is great!

PHP will be used to access user data such as date posted and if the user is a registered user or not. Also PHP will be used to gather the input from the submit form and add it to the mySQL database thus publishing it onto the anime profile.
